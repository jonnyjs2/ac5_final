Tips for using GreenSpace Finder

  • Click any park on the map to open the detail panel with photos, distances, and amenity list.
  • Use the filter button on the map page to show/hide playgrounds, trails, and dog areas.
  • If you see an error in a park's data, check the Sources page to learn how to report it.
